2.En l’arxiu index.html dins de pi_2

2.1.Modifiqueu el CSS per fer que el menú quedi centrat utilitzant GRID de CSS i utilitzi rem en lloc de px

En aquest cas en comptes d'utilitzar simplement rem he volgut fer un div amb classe grid que en el css té la propietat " grid-template-columns: repeat(12,1fr);", de tal manera que cada "button" esta dintre del div "buttons". En el css el div "buttons" té la propietat "grid-column: 5/9;" aixo el que fa és que els buttons ocupin l'espai entre la columnma 5 i 9.
